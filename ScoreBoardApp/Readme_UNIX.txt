Requerimientos:

Instalar las fuentes digital-7.ttf y scoreboard.ttf para mejorar la interfaz del sistema,
el software requiere de java 1.8.0 como minimo para funcionar.

Instalacion:

	Opcional, recomendado: 
	1.- Instalar las fuentes dando doble click sobre los documentos *.ttf.
	
	2.- Establecer una conexión bluetooth con el dispositivo (ScoreBoard) y averiguar el puerto /dev/cu.* asignado a este.
	3.- Abrir el software dando doble click sobre el fichero *.jar, verificar que la carpeta /lib y sus contenidos existan al lado del fichero ejecutable.
		
Uso:
Si el puerto serial /dev/cu.* del dispositivo ha sido asignado, saltar al paso 2.
	1.- Establecer una conexión bluetooth con el dispositivo (ScoreBoard) y averiguar el puerto /dev/cu.* asignado a este.
	
	2.- Abrir el software, en "settings" se encuentra una lista de dispositivos /dev/cu.* disponibles, si su dispositivo no aparece, presione sobre la lupa y espere hasta que la lista se remueve, si aun no aparece reinicie el sistema.
	SI EL PROBLEMA PERSISTE, verifique el paso 1, tal vez el dispositivo aun no ha sido asignado a algun puerto /dev/cu.*.
	3.- Una vez que aparezca la palomita (check) al lado de la lista de dispositivos, usted ya puede utilizar el sistema.
	
	Para enviar un dato presione sobre el boton en forma de disckete.
	para verificar los mensajes del dispositivo presione sobre los lentes.
	Para limpiar el "log" presione sobre el boton en forma de borrador.
